About Aphrozeus

What's the value of public data, if there are no possibilities to easily access it. The blockchain, tangle, hashgraph or whatever technology is used to transfer cryptocurrency is a place full of interesting information. All this data is publicly available, but only as bits and bytes. The mission of Aphrozeus is to transform these bits and bytes into an easily explorable format - and by exploring we don't just mean browsing through transaction histories.

The first step of Aphrozeus is to enable you, to observe transactional data and get alerted in real-time. This can be crucial for trading decisions, risk management or monitoring. We call the functions that observe each action on the blockchain triggers and each one can raise an alarm. Up to now, these triggers are developed and maintained by us. But this is not supposed to stay like this...

Roadmap

The current triggers are maintained and developed by us, but we want to enable you to create and configure your own triggers so you don't miss any important information anymore. To enable this, we've the following elements on our roadmap:

  • Custom trigger creation and configuration
  • Support of different actions if a trigger found a match
    • Email
    • SMS
    • Chat Message
    • Websocket
    • ...
  • Graph mining and machine learning to detect patterns and predict events
  • Search engine for events and transactions
